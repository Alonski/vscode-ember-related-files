## Default keybinding

- **Mac**: <kbd>`alt`</kbd> + <kbd>`cmd`</kbd> + <kbd>`r`</kbd>
- **Win**: <kbd>`ctrl`</kbd> + <kbd>`'`</kbd>
- **Linux**: <kbd>`alt`</kbd> + <kbd>`r`</kbd> + <kbd>`f`</kbd>
